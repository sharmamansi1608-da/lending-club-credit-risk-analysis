# Lending Club Loan Default Risk Analysis

##  Overview
This project analyzes lending data to identify borrower segments with high default risk and evaluate financial impact.

##  Objective
- Identify key factors influencing loan defaults
- Segment borrowers based on risk
- Analyze financial losses across loan categories

##  Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQLite)
- Statistics (T-test)

##  Key Insights
- Default rate ~19.6% across portfolio
- High DTI (>30) significantly increases default risk
- Low-income borrowers (<40K) show higher default rates
- Debt consolidation loans contribute highest financial losses

##  Business Recommendations
- Apply stricter approval for high DTI borrowers
- Focus risk control on mid-income segments
- Use DTI as key decision metric
